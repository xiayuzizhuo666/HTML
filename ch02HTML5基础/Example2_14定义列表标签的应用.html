Oxford Dictionary

HTML
Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages:'an HTML file'.
World Wide Web
An information system on the Internet which allows documents to be connected to other documents by hypertext links, enabling the user to search for information by moving from one document to another.
browser
A computer program with a graphical user interface for displaying HTML files, used to navigate the World Wide Web: 'a web browser'.
